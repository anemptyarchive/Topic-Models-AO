# 青トピックモデル

　『トピックモデル』(MLPシリーズ)の勉強会資料のまとめです。本の補助として読んでください。  

　本を参考にして、以下のモデルに対する推論処理をRやPythonでスクラッチ実装します。

- ユニグラムモデル(unigram model)
- 混合ユニグラムモデル(mixtrue of unigram models)
- トピックモデル(topic model)

　ブログでは、各種モデルやアルゴリズムを「数式」と「プログラム」を用いて解説します。
